# Улучшения Telegram-бота ЭРА

## Реализовано сейчас

| Раздел | Улучшение | Статус |
| --- | --- | --- |
| Админ-панель | Раздел «Управление» стал понятнее: аналитика, ожидания, опросы, цели, организации, структура, права и настройки разделены по смыслу | Готово |
| Аналитика | Добавлен экран «Что где ждёт» для быстрых управленческих решений | Готово |
| Аналитика | Добавлены срезы по статусам заявок, ролям, возрасту, департаментам, направлениям, событиям, проектам, задачам и опросам | Готово |
| Опросы | Раздел опросов остаётся доступен из аналитики и управления | Готово |
| Excel | Сохранены отдельные выгрузки базовой аналитики и опросов | Готово |
| Тесты | Добавлены проверки новых кнопок и срезов аналитики | Готово |

## Рекомендуется следующим этапом

| Раздел | Что улучшить | Почему |
| --- | --- | --- |
| Участники | Сохранённые фильтры и быстрый поиск из любого списка | Для 200+ участников списки станут длинными |
| Аналитика | Экспорт выбранного среза отдельным Excel-файлом | Админу будет проще делиться конкретной статистикой |
| Права | Экран «что даёт каждое право» внутри бота | Снизит риск случайной выдачи лишних доступов |
| FSM | Единая кнопка отмены для всех сценариев ввода | Уменьшит зависания в режимах ожидания текста |

## Необязательно

| Идея | Комментарий |
| --- | --- |
| Дашборд с графиками внутри Telegram | Можно позже, после стабильной эксплуатации |
| Автоматические PDF-отчёты | Полезно, но не критично для запуска |

## Требует изменения бизнес-логики

| Идея | Что нужно решить |
| --- | --- |
| Автоматическое принятие управленческих решений по опросам | Нужны правила совета ЭРА |
| Автоматическая оценка активности департаментов | Нужно определить формулу рейтинга департамента |
